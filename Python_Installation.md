<h2><a id="user-content-installation-instructions" class="anchor" href="#installation-instructions" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation Instructions</h2>
<h3><a id="user-content-windows-setup" class="anchor" href="#windows-setup" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Windows Setup</h3>
<ol>
<li>
<p>Download the <code>python 3 exe installer</code> from <a href="https://conda.io/miniconda.html">https://conda.io/miniconda.html</a> (If you are not sure and you are using windows 7, 8, 9 or 10, then download the 64-bit exe installer)</p>
</li>
<li>
<p>Run the exe installer and install using default choices.  By clicking next.  (The installation might take a few minutes if the computer is slow, you can click "Show Details" to see the installation progress.</p>
</li>
<li>
<p>When installation is complete, find and open a program called <code>Select Anaconda Prompt</code> from the Windows Start Menu.</p>
</li>
</ol>
<p>This will open a new window with a black background (it might keep blinking if the computer is slow).  Wait until you can start typing</p>
<ol start="4">
<li>
<p>Type <code>conda install jupyter pandas matplotlib</code> and hit <code>ENTER</code> key.  When asked do you want to proceed, type <code>y</code> and hit <code>ENTER</code>.</p>
<p>This will take a a few minutes to download the jupyter, pandas and matplotlib packages.</p>
</li>
<li>
<p>Type <code>conda install -c conda-forge basemap=1.0.8.dev0</code> and hit <code>ENTER</code>. Type <code>y</code> when asked if you want to proceed. Once this has finished installing, type:</p>
</li>
</ol>
<p><code>conda install -c conda-forge basemap-data-hires</code></p>
<ol start="6">
<li>
<p>To view a list of packages and versions installed, or to confirm that a package has been added or removed, type <code>conda list</code>. Confirm that jupyter, pandas and matplotlib have been installed</p>
</li>
<li>
<p>The Python session of ResSearch will be using jupyter notebooks. When you open Jupyter, the directory in which you are currently located becomes the "home directory" for Jupyter. To change this home directory (preferably to C:/ drive, or your user folder), in your command line you enter  <code>cd C:\Users\myUsername</code>, or <code>cd &lt;file path to navigate to&gt;</code></p>
</li>
<li>
<p>Type <code>jupyter-notebook</code> and press <code>ENTER</code> key. This will start jupyter notebook in the default browser.  It might open in Internet Explorer.  If you prefer to use Chrome or Firefox, then you need to change the windows settings to use Chrome or Firefox as the default browser. If this does not open automatically, you can copy the "local host" URL on your command line to a new browser tab.</p>
</li>
</ol>
<p>This instance of jupyter notebooks is running through your command line, and closing your command terminal will also terminate your Jupyter session. Alternatively, if you wish to exit Jupyter through the command line, use <code>ctrl-C</code>.</p>
<ol start="9">
<li>If you need to uninstall miniconda for any reason, you can do this through "Add or remove Program" in the control panel, by removing "Python 3.6(Miniconda)"</li>
</ol>
<h3><a id="user-content-os-x-setup" class="anchor" href="#os-x-setup" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>OS X Setup</h3>
<ol>
<li>In your browser download the Python 3.6 Miniconda installer for OS X from <a href="https://conda.io/miniconda.html">https://conda.io/miniconda.html</a>, then in your terminal window type the following:</li>
</ol>
<p><code>bash Miniconda3-latest-MacOSX-x86_64.sh</code></p>
<ol start="2">
<li>
<p>Follow the prompts on the installer screens. If unsure about any setting, simply accept the defaults as they all can be changed later. After installation is complete, close your terminal window.</p>
</li>
<li>
<p>Re-open your terminal window, and type <code>conda --version</code> into the command terminal to test that miniconda has been installed. Conda should respond with the version number that you have installed, like: conda 3.11.0</p>
</li>
</ol>
<p>NOTE: If you see an error message, check to see that you are logged into the same user account that you used to install Anaconda or Miniconda, and that you have closed and re-opened the terminal window after installing it.</p>
<ol start="4">
<li>
<p>Type <code>conda install jupyter pandas matplotlib basemap</code> and hit <code>ENTER</code> key.  When asked do you want to proceed, type <code>y</code> and hit <code>ENTER</code>.</p>
</li>
<li>

If you prefer to use Chrome or Firefox, then you need to change the windows settings to use Chrome or Firefox as the default browser. If this does not open automatically, you can copy the "local host" URL on your command line to a new browser tab.</p>

<p>You may also edit ~/.bash_profile and remove the miniconda directory from your PATH environment variable, and remove the hidden .condarc file and .conda and .continuum directories which may have been created in the home directory with: <code>rm -rf ~/.condarc ~/.conda ~/.continuum</code></p>
